# Лабораторная работа 1
Слонова Анна М8О-101БВ-25 фреймворки Python для анализа данных лабораторная работа 1

## Задача
Освоение базового ML-пайплайна на табличных данных
В рамках данной работы студент должен продемонстрировать понимание и практическое
применение полного цикла машинного обучения на примере специально «испорченного»
табличного датасета. Датасет будет предоставлен преподавателем и будет содержать
типичные проблемы реальных данных: пропущенные значения, выбросы, несогласованные форматы,
дубликаты, логические ошибки (например, отрицательный возраст или нереалистичные значения),
а также избыточные или неинформативные признаки.

Цель: использование линейной регрессии для предсказания CGPA на основе режима занятий
и сна, использование логистической регрессии для предсказания риска развития депрессии
на основе образа жизни.
